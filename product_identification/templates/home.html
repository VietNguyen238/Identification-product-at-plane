Project information

This project is a Django web application that involves real-time video processing using a YOLO model for object detection. The application includes features such as:

  • Real-time Video Streaming: Capturing video from a camera and processing it using a YOLOv11 model to detect objects.
  • Web Interface: Displaying the video stream on a webpage with an option to stop the stream.
  • Image Capture: Capturing and saving frames when certain conditions are met (e.g., object detection confidence is above a threshold).

Confusion matrix

The confusion matrix is a summary table used to evaluate the classification model by comparing the predicted results with the actual outcomes. It provides insights into the model's performance, helping to identify areas for improvement and ensuring accurate product sorting at the airport.

Batch training

Batch training is a method of training a machine learning model where data is divided into batches (small groups) before being fed into the model to update the weights. This approach helps in efficiently managing memory and computational resources, allowing for faster and more stable training processes.

Curve chart

A curve chart is a graphical representation that displays data points connected by a continuous line, illustrating trends or changes over a specific variable, such as time, quantity, or epochs. This type of chart is particularly useful for visualizing the progression and patterns in data, aiding in the analysis and interpretation of trends.

Labels

Labels help the model learn how to predict or classify data based on the information associated with each data sample. They are essential for training machine learning models, as they provide the ground truth that the model uses to learn and make accurate predictions.

Performance index

The performance index refers to the results or predictions of a model, which are compared with actual labels to assess its performance on tasks like classification or regression. This index provides valuable insights into the model's accuracy and effectiveness, helping to identify areas for improvement and ensuring optimal performance in product sorting at the airport.
